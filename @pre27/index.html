Más características
La mesa para portátil es adecuada para muchas ocasiones, soporte de portátil para la cama, una mesa de cama para niños, una mini mesa de escritura, mesa de sofá para portátil, una mesa de pie para el trabajo de oficina
Abra las patas de la mesa para usarla. después de usarla, pliéguela hasta un tamaño compacto y guárdela junto al sofá o debajo de la cama sin ocupar demasiado espacio
Los botones de bloqueo automático en cada lado pueden cambiar de altura fácilmente, las patas se pueden ajustar a 5 alturas diferentes (ajustable de 10.6" - 15.7"), además dos abrazaderas se pueden utilizar para ajustar 4 ángulos diferentes (de 0-35°)
El tamaño del escritorio del ordenador portátil es 23.6*15.7*10.6~15.7" que es bastante para caber todo el tamaño del ordenador portátil. Ranura de soporte incorporada para apoyar tabletas y teléfonos móviles. El cajón puede guardar sus bolígrafos, gafas, ratón y cables USB, etc.
