Autres caractéristiques
La table pour ordinateur portable convient à de nombreuses occasions, support d'ordinateur portable pour le lit, table de lit pour enfants, mini table d'écriture, table de canapé pour ordinateur portable, table debout pour le travail de bureau.
Ouvrez les pieds de la table pour l'utiliser. Après utilisation, pliez-la pour la rendre compacte et la ranger à côté du canapé ou sous le lit sans qu'elle ne prenne trop de place.
Les boutons de verrouillage automatique de chaque côté permettent de modifier facilement la hauteur, les pieds peuvent être réglés à 5 hauteurs différentes (réglables de 10,6" à 15,7"), et deux pinces peuvent être utilisées pour régler 4 angles différents (de 0 à 35°).
Le bureau pour ordinateur portable a une taille de 23,6*15,7*10,6~15,7", ce qui lui permet de s'adapter à toutes les tailles d'ordinateurs portables. La gorge du support intégré permet de soutenir les tablettes et les téléphones portables.
