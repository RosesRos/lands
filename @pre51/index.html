This keyboard is great for typing. I get a lot of momentum while typing (I'm typing this review on it) and the keys have a great reaction. I wish there was maybe a way to lock the keyboard while it's open so you can use it on an uneven surface (like a bed) but that's just a minor hangup (I primarily use this on a desk anyway).

However there are a couple small nags that keep it from perfection. For one thing the keyboard mapping is a little annoying. Escape is accessed by pressing Fn + Shift + `/~, which makes it inconvenient to hit Esc quickly (particularly if you're also holding Ctrl in order to bring up the Start Menu). Esc should have been made a separate key from `/~. Also Delete is accessed via pressing Fn and the +/= key, which is also shared with F12 by holding down Shift and Fn. This is annoying because you can't perma-delete things from Windows Explorer because pressing Fn, Shift, and +/= will give you an F12 instead of a Shift + Del (luckily the system I'm using this with has its own internal keyboard so I can use its Delete key for this function). This should really have been mapped as the Fn function for the Backspace key.
